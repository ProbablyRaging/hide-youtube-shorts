Turbo Mode

Turbo mode allows HYS to be more aggressive in the way that it hides Shorts, at the cost of a little more system resource usage

Enable turbo mode under Miscellaneous > Turbo

Mobile Compatibility

HYS now works for the mobile version of YouTube. Download Kiwi Browser on Android devices and install your favorite Chrome extensions
